# CCP Curriculum Map

## Level I — Digital and Algorithmic Foundations

Goal: understand what computing is, how files and commands work, how programs are written, and how progress is archived.

1. What is a Computer Program?
2. Files, Folders, Paths and Working Directories
3. Terminal vs Command Prompt
4. Algorithms and Pseudocode
5. Variables, Values and Expressions
6. First Python Scripts
7. First R Scripts
8. First Git Repository and Course Archive

## Level II — Programming Foundations

Goal: learn decisions, repetition, functions, collections, strings, files, and debugging.

1. Conditions
2. Loops
3. Functions
4. Lists, Vectors and Arrays
5. Strings and Biological Sequences
6. File Reading and Writing
7. Debugging and Error Interpretation
8. Mini-Project 1 — Command-Line Runnable Script

## Level III — Data and Scientific Computing

Goal: handle scientific data, clean tables, compute summaries, plot results, and build reproducible scripts.

1. Tables, CSV and TSV Files
2. Data Cleaning
3. Descriptive Statistics
4. Plotting and Visualization
5. Randomness and Simulation
6. Biological Sequence and Genotype Data
7. Reproducible Scripts
8. Mini-Project 2 — Small Scientific Analysis

## Level IV — Computer Science and Software Construction

Goal: move from scripts to structured software, algorithms, data structures, testing, and maintainable projects.

1. Modular Programming
2. Object-Oriented Programming
3. Data Structures
4. Algorithms and Complexity
5. Recursion
6. Testing
7. Error Handling
8. Mini-Project 3 — Small Software Application

## Level V — Systems and Low-Level Computing

Goal: understand how programs run closer to the machine: compilation, binary representation, memory, pointers, stack, heap, and Assembly.

1. Compilation and Execution
2. Binary, Hexadecimal and Representation
3. Memory Model
4. Pointers and References
5. Stack and Heap
6. Assembly Basics
7. C++ to Assembly Intuition
8. Mini-Project 4 — Low-Level Computation Demonstration

## Level VI — Research-Grade Computing

Goal: build independent research-computing capacity: pipelines, bioinformatics files, reproducibility, validation, documentation, and GitHub publication.

1. Research Project Structure
2. Command-Line Pipelines
3. Bioinformatics File Handling
4. Simulation and Modelling
5. Data Validation and Reproducibility
6. Documentation and Reports
7. GitHub Publication
8. Final Integrated Research Project
