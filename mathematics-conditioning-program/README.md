# Mathematics Conditioning Program (MCP)

The Mathematics Conditioning Program is a research-oriented, university-level applied mathematics conditioning program for mathematical maturity, symbolic fluency, proof competence, computational fluency, modelling ability, scientific communication, and research readiness.

## Core Chain

Mathematical Object → Symbolic Language → Transformation → Proof / Check → Computation → Interpretation → Scientific Communication → Git Archive → Research / Application Use

## Repository Sections

- `course-prompt.md` — operational course prompt and rules
- `course-structure.md` — hierarchy, delivery protocol, grading/logs
- `curriculum.md` — 6-level curriculum map
- `sessions/` — exercises, submissions, attempts, scripts
- `archive/` — correction reports, grade logs, mistake logs, session summaries

## Rule Summary

- Each MCP Session should be a new conversation inside Work Folder: `Mathematics Sandbox` whenever possible.
- The 4 Mandatory Core exercises must always be challenging, teaching, and substantial enough to preserve and advance skills.
- Adult low-tier work must be meaningful, clear, and non-childish.
- Low-energy sessions still require one real progress unit.
- Prerequisite gaps are rescued, not punished.
- Hardest/THE Hardest tasks are assigned by evidence, not ego.
