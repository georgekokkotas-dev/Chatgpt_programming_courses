# MCP Course Structure

## Program Spine

6 Levels × 8 Lessons × 4 Sessions = 192 core sessions.

Progress is based on demonstrated mastery, not simple completion.

## Levels

I. Basic — mathematical language, arithmetic, elementary algebra, geometry, habits.
II. Philosophy — logic, proof philosophy, sets, relations, functions, formal reasoning.
III. Moderate — calculus, analytic geometry, linear algebra, probability, statistics.
IV. High — differential equations, numerical methods, abstract algebra, scientific computing, modelling.
V. Advanced — topology, differential geometry, advanced analysis, optimization, higher structures.
VI. Research Grade — research mathematics, mathematical biology, genomic modelling, proof writing, independent research.

## Lesson Session Pattern

Each Lesson has four Sessions:

1. New concepts
2. Guided practice
3. Mixed symbolic/computational/applied conditioning
4. Review, GitHub/research portfolio task, mini-assessment, mastery evaluation, archive

## Permanent Active Blocks

Mathematical Language; Logic; Sets; Relations; Functions; Proof Theory; Proof Methods; Numbers; Arithmetic; Divisibility; Algebra; Polynomials; Equations; Inequalities; Euclidean Geometry; Analytic Geometry; Vectors; Limits; Derivatives; Integrals; Sequences and Series; Linear Algebra; Probability; Statistics; Differential Equations; Numerical Methods; Abstract Algebra; Topology; Differential Geometry; Mathematical Modelling; Scientific Computing; Programming Commands; Proof Writing; Scientific Writing; Mathematical Biology; Genomic Modelling.

## Exercise Universe

Difficulty levels:

THE Easiest; Easiest; Easier; Very Easy; Easy; Pre-Moderate; Moderate; Post-Moderate; Hard; Very Hard; Harder; Hardest; THE Hardest.

Exercise types:

- D Manual Drill
- L Manual Learning
- C Computational Command Learning
- A Computational Application

## Session Scoring

- Completion % = Mandatory Core only
- Mastery % = correctness, reasoning, fluency, computation, proof quality, communication
- Coverage % = Exercise Universe exploration
- Passing Mastery = 65%
