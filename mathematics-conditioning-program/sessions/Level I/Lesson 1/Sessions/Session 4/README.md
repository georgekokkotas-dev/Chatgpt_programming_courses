## Major Corrections

- `P` or `f` names a function; `P(t)` or `f(a)` is an evaluated function value or dependent quantity.
- A symbol held fixed during one model evaluation is normally a parameter, not the independent variable.
- Genotype frequencies and proportions are dimensionless; population counts are measured in individuals.
- `f(4)` is a function value, not the function itself.
- Cancellation in `ab/a=b` requires `a ≠ 0`.
- Python function definitions require a colon: `def formula(h0, k, t):`.
- The model `G(t)=G0(1+rt)` must be implemented with addition inside the factor, not multiplication alone.
- A publication-style model description must state a usable domain, coherent units, assumptions, interpretation, and a clearly labelled illustrative validation case.
