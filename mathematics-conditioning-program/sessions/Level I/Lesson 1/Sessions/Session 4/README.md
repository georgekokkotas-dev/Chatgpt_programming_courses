## Major corrections

- `f` is the function; `f(a)` is the value produced at input `a`.
- A symbol fixed during one model evaluation is generally a parameter.
- Frequencies and proportions have no physical units.
- Cancellation in `ab/a = b` requires `a ≠ 0`.
- Clean set-builder notation is `{x ∈ S | condition}`.
- In Python, `x = 5` assigns, `x == 5` compares, and a function signature ends with `:`.
