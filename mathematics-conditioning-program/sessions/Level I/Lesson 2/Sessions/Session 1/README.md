## Major corrections

- In this course, `N = {0,1,2,3,...}`; infinity is not an element of `N`.
- `-17 ∈ Z`, `14/7 = 2 ∈ N`, and `√49 = 7 ∈ N`.
- A valid counterexample must satisfy the premise and contradict the conclusion; `1/2` disproves “every rational number is an integer.”
- `x ∈ A` is element-to-set; `A ⊆ B` is set-to-set.
- `4 + 0i = 4 ∈ R`; a non-zero imaginary part is what makes a number genuinely non-real complex.
- Population change is naturally integer-valued, mutant-cell proportion belongs to `[0,1]`, and a general Fourier coefficient may belong to `C`.
- Division closure always excludes a zero divisor.
