M1 — Mathematical Language / Logic / L / Easy–Pre-Moderate
For each item, classify it as:
A) mathematical statement,
B) mathematical expression,
C) condition/predicate,
D) non-mathematical subjective sentence.

Items:
1. x + 3 B
No absolute truth value, thus x can be anything. 

2. x + 3 = 10 A
Since it is equal to 10, there's an absolute truth value, but x has to be defined, to be either TRUE or FALSE. 
But it is a statement, since it basically says "some number plus 3 equals exactly 10".

3. 7 > 2 A
It is a statement. It has an absolute truth value, it holds TRUE and it states something very precisely and clearly: "7 IS larger than 3"

4. x ∈ R A
This is a statement. No debate, there is an absolute truth value, x has to be defined clearly before T of F is stated or proved, but it is a statement.
x BELONGS in R! Also, can be a Condition.

5. This function is elegant. D
This is a personal opinion/commentary. Not a mathematical statement or expression.

6. If x > 4, then x > 1. C
Could be a predicate, could be a condition. If we say "for x to be less than 1, x ALSO has to be less than 4", it is a condition.
If we say "if x is more than 4, then it is more than 1 TOO", that's a P => Q predicate and in this case it is TRUE. (Corrected)

For every item, give one sentence explaining your classification.

M2 — Implication Structure / Logic / D+L / Easy–Pre-Moderate
For each implication, identify:
- hypothesis P
- conclusion Q
- whether the implication seems true or false over R
- if false, give one counterexample

Implications:
1. x > 5 ⇒ x > 0 TRUE
2. x > 0 ⇒ x > 5 FALSE, Could be 4.999
3. x = 2 ⇒ x² = 4 TRUE 
4. x² = 4 ⇒ x = 2 FALSE, Could be x = -2
Hypotheses/Assumptions (P(x)) are the equalities/inequalities before the "⇒" symbol.
Conclusions (Q(x)) are the equalities inequalities after the "⇒" symbol.

M4 — Function Language / Functions + Scientific Writing / L / Pre-Moderate
Consider the function statement:

f: R → R,   f(x) = 2x - 1

Answer:
1. What is the input variable?
It is a real number

2. What is the output expression?
It is also a real number, that takes the x input and converts it to a y value defined by the function.

3. What does f(3) mean in words?
It means that, if we double 3 and subtract 1 from it, we'll get an (x,y) point of the f function that corresponds to our input. In this case (3, f(x)) = 5

4. Is f(x)=2x-1 a statement, expression, rule, or equation? Defend your classification.
It is not an equation. An equation has an exact value.
It is not a rule, it is comprised of rules, but it isn't one.
It is not an expression. It has a result.
It is a statement. It states that, this function gives a specific y value for every x assigned to it.

5. Write one clean scientific sentence describing what the function does.
It takes an x value, doubles it, subtracts 1 and gives an (x, y) point in the chosen coordinate system.

R3 — Equality vs Implication / Proof Literacy / L / Pre-Moderate
Explain the difference between these two lines:

A) x = 3
This is an equation. It can't be disputed in such form. x = 3.

B) x > 5 ⇒ x > 3
This is a predicate. We're assuming, and quite correctly here, that, since x is greater than 5, it is also greater than 3.

= means EQUAL! If we're talking Truth values, we can start asking questions. But it means Equal. 
This ⇒ on the other hand, is an implication. We're not sure, it's not definite.
But we shouldn't confuse it with this ∴, though

Your answer must include:
1. what kind of mathematical object each line is,
2. whether each can be true or false,
3. why = and ⇒ cannot be used interchangeably.

RESEARCH INTEGRATION — 2 exercises

RI1 — Genomic Classification Logic / Mathematical Language + Research Writing / A / Pre-Moderate
Consider the research-style implication:

If variant V satisfies pathogenicity criteria C, then V belongs to the set P of clinically actionable variants.

1. Write this implication symbolically.
V ∈ C => V ∈ P  
2. Identify the hypothesis.
P: V ∈ C
3. Identify the conclusion.
Q: V ∈ P
4. State one reason why the definition of C matters.
Because we can't arbitrarily decide when a variant is pathogenic, unless we take serious considerations of what pathogenicity is in genomic science.
5. State one reason why this is not yet a biological proof of causation.
Pathogenicity doesn't mean you can do anything about it, nor a mathematical/logic formula can decide whether a variant is pathogenic, nor can we imply that, meeting the criteria is the same as being pathogenic.
