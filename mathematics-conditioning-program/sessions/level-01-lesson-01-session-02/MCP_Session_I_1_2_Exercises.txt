M4 — Function Rule vs Value / Functions + Mathematical Language / L / Pre-Moderate
Consider:

f: R → R,   f(x) = 3x + 2

Answer:
1. What is the function rule?
Every point of the input value belongs to R, every function's output belongs to R
2. What is the input variable?
x
3. What is f(4)?
f(4) = 3 * 4 + 2 = 14
4. What point on the graph corresponds to input 4?
(4, 14)
5. Is f(x)=3x+2 the same kind of object as f(4)=14? Explain the difference.
No. f(x) is the function's formula, where f(4)=14 is a statement about its value on the point (4, y)
